# Support

## Getting help

- Usage questions and feature requests: GitHub Issues
- Documentation: https://phodal.github.io/routa/
- Security issues: [SECURITY.md](SECURITY.md)

## Before opening an issue

- Check the README and docs site first.
- Include environment details, reproduction steps, and logs when relevant.
- For desktop issues, specify OS and whether the issue is in web, desktop, or both.
